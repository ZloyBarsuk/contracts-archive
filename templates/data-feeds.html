Data Download

This page contains the data behind the Contracts Finder Archive which can be downloaded in XML, CSV and SQLite database format.

Terms and conditions apply to the use of this data. See the bottom of this page for details.

The current Contracts Finder site is available here. Data from the current site is available in Open contracting Data standard format. For guidance on how to access this please click here

Database download

An SQLite database file of all of the contract information is available to download.

  • Database of all contracts (90Mb)

Activity summary and buyer group list

You can download three reports. The first provides a high-level view of procurement activity for all government departments and agencies that have published notices. It shows the total number of tender document summaries, contract document summaries and live opportunities published so far by each organisation.

The second is a list of buyer groups registered on Contracts Finder is also available.

Thirdly, a list of the pipelines that government is currently aware of that could result in contract opportunities.

They are in CSV format which can be opened as a spreadsheet or a simple text file.

They are no longer updated daily and contain the information present when the site was made read only.

  • Summary of activity by department
  • List of buying organisations
    • List of current pipelines

Detailed reports - recent notices

Detailed contract information is available in XML and CSV formats.

The CSV version includes - for each notice - its title and brief description, the name of the buying organisation and the contract value. The machine-readable XML document contains additional data for each notice, including CPV codes.

They are no longer updated daily and contain the information present when the site was made read only.

  • Detailed report - recent notices (XML)
  • Detailed report - recent notices (CSV)
